# 交互

- 藏品展柜（中间闪光特效，代表有藏品）
    玩家可以攻击，攻击之后，变成碎掉的展柜（不可交互），显示msg，

- 碎掉的展柜
    第一次靠近会触发藏品丢失事件，右上角增加黑色藏品图标，代表这个藏品丢了（如果获得藏品，会变成彩色）

- 地图
    有地板/墙

- 三种药
    分别对应不同的回复血量，玩家捡起之后，回血

- 陷阱
    玩家接触之后，产生眩晕效果，扣血，同时冻结行动若干秒。
    陷阱触发次数可配置（1次/N次），次数用完之后消失

- 容器
    对攻击无反应，可交互，交互之后变成打开状态
    交互结果1: 生成老鼠
    交互结果2: 获得藏品 / 钥匙
    交互结果3: 什么都没发生

- 钥匙 （场景关卡唯一）
    携带钥匙后，靠近门，可以打开门，进入下一关

- 门
    见钥匙，玩家不带钥匙时，不可通过，携带钥匙后，靠近门自动打开，可通过

- 通关点
    玩家进入通关点之后，进入下一关

- 老鼠
    - 生成老鼠后，老鼠会停顿若干秒
    - 老鼠的AI是逃离玩家
    - 玩家可以攻击老鼠，老鼠会死亡，留下一滩血迹

- 贼
    - 贼的朝向（只有左右）发出一条检测射线（会被障碍物挡住）检测玩家，检测到之后【看到玩家】
    - State Idle: 贼随机走动，【看到玩家】之后进入Escape状态
    - State Escape: 初始方向v为远离玩家的方向，进入弹球模式，经过N秒之后，回到Idle模式。或者玩家在Idle状态或者Escape碰到或攻击贼、或者踩到陷阱（总之就是贼掉血），进入Rage状态
    - State Panic: 沿着地图中间的柱子顺时针巡逻，当经过M秒后，进入Escape2状态。当看到玩家时，进入Rage状态。
    - State Rage: 先停顿M秒（表现效果可以为红眼），确定v为接近玩家的方法，然后以v速度移动，N秒停下，移动过程中，碰到障碍物，进入眩晕状态（不掉血）。碰到陷阱掉血血晕。碰到玩家，玩家掉血。N秒结束之后，暂停X秒进入Panic状态。
    - State Escape2: 门打开，贼逃走，进入门内。

# 设定

- 血量
    - 玩家血量归零失败，进入结局。

- 受伤
    - 受伤之后，进入N秒的无敌状态（表现效果可以闪烁呈现）。

- 眩晕状态
    - 冻结实体行动N秒

- 结局画面
    - 不同的条件，显示不同的文字
    - 可会到主菜单
